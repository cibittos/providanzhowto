STEP 1

CAPTURE VIDEOS ACCORDING TO PROJECT

Firstly, decide the project type, number of stations and number of trials for each station and capture videos (preferrably in .mp4 format)

STEP 2

START NEW PROJECT ON PROVIDANZ

Open browser (Chrome, Firefox & Safari only) and start a NEW PROJECT and enter project details including the details used to capture the videos

STEP 3

ENTER PROJECT SETTINGS & STATION DETAILS

Enable or disable task category and value-add analysis options, and enter all the required settings

STEP 4

ANALYZE THE PROCESS VIDEOS

Select video files, play them, get timestamps, and enter details for the task element happening between the timestamps.

STEP 5

VIEW DATA TABLES AND CHARTS

The application displays station-wise data tables, range charts and composite stacked bar charts for the entire assembly line

STEP 6

DOWNLOAD DATA & DOCUMENTS

Click on document type links to download data tables, Standard Work Instructions and Kaizen Tables
